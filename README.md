# 🏛 YojanaSarthi
AI-Powered Government Scheme Eligibility & Matching Platform

YojanaSarthi is a smart GovTech platform that helps citizens discover and check eligibility for government schemes based on their personal profile.

The platform uses an intelligent rule-based eligibility engine to match users with relevant schemes instantly.

🚀 Problem Statement

Many citizens in India are unaware of government schemes they are eligible for.
Existing portals are:

Information-heavy

Difficult to navigate

Not personalized

Lack eligibility clarity

YojanaSarthi solves this by providing:

✔ Personalized eligibility checking
✔ Intelligent scheme recommendations
✔ Clean and user-friendly UI
✔ Profile-based filtering

🧠 Solution Approach

The system uses a structured eligibility engine that evaluates:

Age

Income

Category

Occupation

Scheme requirements

It calculates:

Eligibility status

Recommendation score

Highly recommended schemes

This avoids heavy ML models, making the system:

Lightweight

Scalable

Cost-efficient

Government-deployment friendly
